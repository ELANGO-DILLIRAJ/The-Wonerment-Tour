# The Wonderment Tour: Global Odyssey 🌍🎵

A premium, minimalist landing and registration web application dedicated to the iconic live world tour of A.R. Rahman. Designed with a sharp focus on editorial typography, clean visual space, and real-time interactive mechanics.

## 🎨 Project Overview
This project completely reimagines traditional concert booking platforms. Moving away from cluttered structures, it embraces a high-end, luxury design system (utilizing pure black, crisp white, and deep architectural slate lines) that puts the focus entirely on the storytelling narrative.

### Key Promotional Elements Integrated:
* **The Hook:** *Symphony of the Soul* — A grand orchestral journey live across continents.
* **The Roots:** *From the Mozaic of Madras to the Global Stage* — Dedicated timeline chapters tracking legacy legs in Chennai and Mumbai.
* **The Atmosphere:** *Infinite Echoes* & *Rhythms of Reality* — Evocative copy sections that outline the visual and sonic tech powering the live arenas.

---

## 🛠️ Features & Technical Mechanics

* **Minimalist Architecture:** Interface styled purely using native CSS layouts, ditching bulky frameworks for high performance and fast loading speeds.
* **Dynamic Interaction Loop:** A real-time ticket price processing system engineered with Vanilla JavaScript using `input` and `change` event listeners.
* **Native CSS Typewriter Animation:** The opening tagline smoothly types itself out character-by-character on page load using fluid CSS keyframe steps.
* **Smart Geographic Routing:** The form is structurally mapped out to route global users to their native ticketing platforms (e.g., BookMyShow/District for India, Platinumlist for Dubai, AXS for London, and Ticketmaster for New York).
